Rouse 更新日志

最后更新：2026 年 4 月 26 日

这里记录 Rouse Mac App Store 版本的发布说明，以简洁、面向用户的方式展示公开版本历史。

1.7.0

新增官方 rouse 命令行工具。

  • 新增内置 rouse CLI，可用于 Terminal、shell 脚本和本地自动化流程。
  • 命令行可以启动、停止、查看状态、打开 Dashboard，并控制显示器防睡眠设置；会话仍由 Rouse.app 持有。
  • 在设置中新增命令行引导，说明权限用途和拒绝后的恢复方式。
  • 改进了菜单栏打开 Dashboard 的体验，并打磨 About Rouse 中的支持操作。

1.6.0

改进了全应用的辅助功能支持。

  • 在 Dashboard、Quick Panel、Settings、About 和菜单栏控制中提供更清晰的 VoiceOver 标签、数值和提示。
  • 改进 Quick Panel、Settings 和 About 的键盘导航。
  • Dashboard 现在会更好地响应 macOS 显示辅助功能设置，包括 Reduce Motion、Reduce Transparency、Increase Contrast 和 Differentiate Without Color。
  • 调整菜单栏图标对齐。

1.5.0

现在可以在会话进行中直接调整显示器常亮设置。

  • 无需停止当前会话，也可以切换显示器常亮设置，只在需要时保持屏幕点亮。
  • 同时改进了发布流程相关的稳定性和会话处理的可靠性。

1.4.0

进一步打磨 Dashboard 的布局体验。

  • 优化 Dashboard 的布局与间距，让整体界面更简洁、更从容。
  • 改进较长内容场景下的可读性与视觉平衡。

1.3.0

新增 App Shortcuts，并进一步打磨 Dashboard 体验。

  • 新增 App Shortcuts，可直接开始会话、开始定时会话、停止会话，或打开 Dashboard。
  • 优化 Dashboard 的状态切换和布局稳定性，长文案和多语言场景下也更清晰。
  • 补充了更直接的反馈与支持入口，并更新了更易识别的菜单栏图标。

1.2.0

新增日语和中文本地化支持。

  • 新增日语、简体中文和繁体中文本地化。
  • 更新 Dashboard 与 Quick Panel 文案，使其与当前产品表达保持一致。
  • 改进 Dashboard 定位行为与会话状态稳定性。

1.1.0

新增可选的打赏支持。

  • 在 About Rouse 窗口中加入了 “Support Rouse” 区块，提供打赏选项和 Restore Purchases。
  • 改进了 App Store Connect 审核素材管理的内部工具。

1.0.0

Mac App Store 首次发布。

  • 提供菜单栏快速防休眠控制，以及用于更深入控制的简洁 Dashboard。
  • 支持不限时和定时会话、显示器睡眠控制、电池保护和登录时启动。
  • 支持在重新启动应用或系统唤醒后恢复定时会话。
