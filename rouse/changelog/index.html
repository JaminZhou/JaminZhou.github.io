Rouse Changelog

Last updated: April 28, 2026

Release notes for the Mac App Store version of Rouse. This page keeps the public version history short and user-focused.

1.7.0

Added the official rouse command line interface.

  • Added a bundled rouse CLI for Terminal, shell scripts, and local automation workflows.
  • Let command line tools start, stop, inspect status, open the Dashboard, and manage display sleep while Rouse.app remains the session owner.
  • Added Command Line setup guidance in Settings, including permission context and recovery steps.
  • Improved Dashboard access from the menu bar and polished support actions in About Rouse.

1.6.0

Improved accessibility across the app.

  • Added clearer VoiceOver labels, values, and hints in Dashboard, Quick Panel, Settings, About, and the menu bar control.
  • Improved keyboard navigation in Quick Panel, Settings, and About.
  • Made Dashboard respect more macOS display accessibility settings, including Reduce Motion, Reduce Transparency, Increase Contrast, and Differentiate Without Color.
  • Refined menu bar icon alignment.

1.5.0

Change display sleep while a session is already running.

  • Let you adjust display sleep while a session is active, so you can keep the screen awake only when you need it.
  • Included behind-the-scenes reliability improvements for the release flow and session handling.

1.4.0

Polished the dashboard layout.

  • Refined dashboard layout and spacing for a cleaner, calmer presentation.
  • Improved readability and balance for longer dashboard content.

1.3.0

Added App Shortcuts and refined the release experience.

  • Added App Shortcuts for starting a session, starting a timed session, stopping a session, and opening the Dashboard.
  • Refined Dashboard transitions and layout behavior for longer and localized text.
  • Added clearer feedback and support entry points, and updated the menu bar icon.

1.2.0

Added Japanese and Chinese localization support.

  • Added Japanese, Simplified Chinese, and Traditional Chinese localizations.
  • Refreshed Dashboard and Quick Panel copy to match the current product language.
  • Improved dashboard placement behavior and session status reliability.

1.1.0

Added optional tip jar support.

  • Added a "Support Rouse" section in the About window with tip jar purchases and Restore Purchases.
  • Improved internal tooling for App Store Connect review asset management.

1.0.0

Initial Mac App Store release.

  • Introduced quick menu bar wake controls and a calm dashboard for deeper control.
  • Added indefinite and timed wake sessions, display sleep control, battery protection, and launch at login.
  • Included timed session restore behavior after relaunch or system wake.
